Here is the demo scene for the Moon Tutorial project which involves using TerraLand Terrain component in order to use own custom
heightmap data files from external sources to create and edit high resolution terrain using TerraLand in Unity.

The goal is to show users how to use any custom surface to create terrains in Unity by utilizing TerraLand tools.
